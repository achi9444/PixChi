# 拼豆副業平台網站架構 + UI Wireframe 生成 Prompt

## 目標
我正在開發一個副業產品：
- 核心是「通用拼豆圖紙生成工具」：任何圖片（像素圖或實景照片）都可以生成拼豆圖紙。
- 最終目標是建立「工具 + 創作者市場 + 交易平台」。
- 副業目標：能穩定帶來收入，創作者 PRO 訂閱 + 平台交易抽成。

請根據以下需求，生成完整網站架構與 UI Wireframe 設計建議，包含：

1. 首頁
2. 工具頁
3. 拼豆圖庫
4. 教學文章
5. 創作者專頁（展示）
6. 創作者後台（作品管理、接單管理、收入統計）
7. 客戶訂單流程頁面（建立訂單、付款、出貨、評價）
8. SEO 及流量設計提示
9. 流程圖/用戶體驗路徑

---

## 核心功能需求

### 工具功能（Phase 1）
- 圖片上傳 → 自動生成拼豆圖
- 顏色統計（每個顏色顆數、總顆數）
- 尺寸計算（拼豆格數、實際尺寸）
- 拼豆品牌支援（Perler、Artkal、Hama）
- 顏色數量限制（限制 20~30 色）
- 拼豆底板建議
- 價格估算（顆數 × 單價 + 大型作品加價）
- 下載圖紙（PNG / PDF）

### 創作者系統（Phase 2）
- 創作者專頁：
  - 頭像、介紹、創作風格、作品集、價格範圍、接單按鈕
- 報價規則：顆數單價、最低價格、大型作品加價
- 創作者後台：
  - 作品管理、接單管理、收入統計、出貨管理
- 防止私下接單：訂單前無法下載圖紙，禁止分享私人聯絡方式，平台保障（付款保護、申訴、退款）

### 交易系統（Phase 3）
- 訂單建立、付款、出貨、評價
- 金流串接（綠界科技 ECPay / 藍新金流 NewebPay）
- 平台抽成 5~10%

### SEO / 流量設計
- 關鍵字：
  - 英文：perler bead pattern generator、pixel bead generator、bead pattern maker、image to bead pattern
  - 中文：拼豆圖產生器、圖片轉拼豆、拼豆圖紙生成、拼豆圖工具
- 網站結構：
  - 首頁
  - 工具頁 (/bead-pattern-generator)
  - 拼豆圖庫（Pokemon、Mario、動物…）
  - 教學文章（拼豆新手教學、顏色搭配、尺寸計算）
  - 創作者專區

### 流程圖 / 用戶體驗
- 工具入口 → 生成圖紙 → 客戶可選擇創作者 → 建立訂單 → 付款 → 創作者出貨 → 客戶評價
- 流程要清楚呈現：工具是入口，創作者專頁是選擇市場，交易是平台保障

---

## UI / UX 設計要求
- 首頁：展示工具入口、熱門創作者、最新圖紙作品、SEO 教學文章
- 工具頁：上傳圖片 → 預覽拼豆圖 → 顏色統計 → 尺寸與價格 → 下載 / 發送給創作者
- 拼豆圖庫頁：圖庫分類（主題、顏色、尺寸）、可搜尋
- 創作者專頁：作品集、介紹、接單按鈕、評價
- 創作者後台：簡單儀表板、訂單列表、收入統計
- 訂單頁：明確顯示圖紙、價格、付款、出貨與評價流程
- 色彩、UI風格：清爽、簡潔、工具友好，突出生成圖紙的功能
