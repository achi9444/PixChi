# PixChi Feature Roadmap

## 目標
- 強化「做得快、做得準、做得出來」的核心體驗。
- 建立一般版與 Pro 版的清楚差異。
- 提升作品展示與分享的吸引力，增加自然擴散。

---

## P0（近期優先）
## 1. 功能力（核心）
- 施工模式穩定化
  - 任務清單、焦點同步、完成標記、覆蓋開關。
  - 目標：降低漏放、提升長圖施工效率。
- 大圖編修流程完善
  - 全圖/分塊切換、分塊內可精修、操作範圍清楚。
  - 目標：大圖也能可編輯、可落地實作。
- 匯出一致性
  - PDF/CSV 匯出前檢查（字型、頁面、空資料）。
  - 目標：減少匯出失敗與格式錯誤。
- 裁切與轉換流程
  - 上傳先預覽，裁切後再轉換。
  - 目標：避免白邊、節省轉換成本。

## 2. 商業賣點（立即可感）
- Pro 施工模板
  - 規則模板 + 手動色序辨識 + 跨作品套用。
  - 目標：凸顯 Pro 生產力優勢。
- 報價體系可用化
  - 一般版顯示固定預設邏輯；Pro 開放調整參數。
  - 目標：一般用戶快速估價，專業用戶可控利潤。

## 3. 可看性（基礎）
- 原圖 vs 轉換圖對比
  - 快速滑桿或切換視圖。
  - 目標：讓成果差異可視化。

---

## P1（中期）
## 1. 功能力（延伸）
- 施工模式進階
  - 規則相似度可視化、一步套用最佳規則。
  - 目標：降低手動排序負擔。
- 操作歷史可視化
  - Undo/Redo 顯示摘要，快速回溯。
  - 目標：減少誤操作成本。
- 自動存檔 UX
  - 顯示最後儲存時間、版本資訊與差異摘要。
  - 目標：提高安心感與可追溯性。

## 2. 商業賣點（Pro 升級）
- 雲端專案（需後端）
  - 專案保存、版本回溯、跨裝置接續。
  - 目標：建立 Pro 長期價值。
- 品牌輸出
  - 匯出附店名/Logo/客戶資訊。
  - 目標：支持商用交付。

## 3. 可看性（互動）
- 轉換過程動畫
  - 像素化、降色、對應色庫過程展示。
  - 目標：提升停留與分享率。
- 作品卡分享
  - 自動生成封面圖（尺寸、顆數、色號數）。
  - 目標：社群擴散。

---

## P2（長期）
## 1. 功能力（平台化）
- 後端 API + 權限完整化
  - 官方色庫後端化、角色權限、審計紀錄。
- 協作流程
  - 定稿者/施工者分工、批註與驗收流程。

## 2. 商業賣點（生態）
- 訂單與報價流程整合
  - 專案 -> 報價 -> 交付 -> 追蹤。
- 高階分析
  - 色號使用熱度、工時預估偏差、利潤報表。

## 3. 可看性（品牌）
- 作品展示牆
  - 風格標籤、難度標籤、熱門模板。
- 成品情境模擬
  - 相框/桌面/牆面預覽。

---

## 一般版 / Pro 分層建議
- 一般版
  - 基本施工模式、完成標記、覆蓋開關、基礎報價顯示。
- Pro
  - 規則模板、手動色序辨識、模板管理、進階報價、雲端專案（後端完成後）。

---

## 成功指標（建議）
- 任務完成效率：施工平均時間下降。
- 匯出成功率：PDF/CSV 失敗率下降。
- Pro 轉換率：模板/報價進階功能使用率提升。
- 分享率：作品卡匯出與分享次數提升。
